Portfolio

Web App
A workout app, for you who needs motivation and inspiration to get started training. The app will help you put together different types of programs, so that the user gets must value out of each traningsession. The main purpose with this is to make the user experience as manageable and intuitive as possible.

After finishing this app I got a deeper understanding of, how you can create value for the user with the help of pertinent functions and cool features. We made use of a platform called Firebase and with the help from an API, we were able to drag content from Firebase into our web app.
Ecooking
Ecooking is a skincare company, that creates natural and organic products. I developed this website for Ecooking's users, so that they could get information and guidance about the products and what they are made of.

My main focus area with this website, was to create beautiful content which matched Ecooking's visual identity. This is expressed in the many illustrations I made for the website. Because Ecooking at that time was a very new company on the marked, I also those to recreate their logo. The idea with my new logo, was to illustrate the story behind the company. Ecooking’s CEO started by testing ingredients home in her kitchen, which lead to her success. Therefore, I made a logo in Illustrator, which represents a pot.
Chicago Roasthouse
We made a collaboration with the take-away restaurant Chicago Roasthouse. The purpose was to optimize their website and create a more manageable user journey.
One of our main focus areas was to design a new visual identity filled with informing illustrations.

Among other I created the infographic underneath the video on the front page. The purpose of this infographic is to show the users how fast and easy it is to order take-away at Chicago Roasthouse. Besides that, we spend a lot of time on creating delicious content for the users, which is expressed in our video on the front page.
U-days & SoMe
In this project we developed a podwalk app for the future students. The purpose with this app is to help the user find a potential education. The app will give the user information about each education on u-days, which hopefully will lead the user to a decision on what they want to study.

I designed a new logo in illustrator for the app, where I used the blue and orange, which already was a part of u-days visual identity. That way I create something recognizable to the user Social Media was also a main target, to market u-days. I made an Instagram competition, which should create attention to u-days and make more people participate.
